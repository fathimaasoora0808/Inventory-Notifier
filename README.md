# Inventory-Notifier
Inventory Notifier is a system designed to store and manage inventory details efficiently. It allows users to add, update, and manage inventory items in a simple and organized way, helping businesses keep track of stock levels accurately.

Objectives:
  • Develop a desktop-based Inventory Reminder System to help small clinics, pharmacies, and stores track inventory efficiently.
  • Provide automated alerts for low-stock and near-expiry items.
  • Create an easy-to-use interface with simple navigation.

Scope:
  • Functional: 
  1.Adding, editing, and deleting inventory items.
  2.Automatic alerts.
  3.displaying inventory in a user-friendly grid.

  • Non-functional:
  1.	Simple UI design.
  2.	Quick startup checks.
  3.	Local data storage.

     
Technology Stack
• Frontend: Windows Forms (C#)
• Backend: C# .NET Framework
• Database: Local storage (extendable to file/database system)
• Cloud/Hosting: Not applicable for desktop-based version
